Current status

For now, fleebmarket is a frontend the /r/mechmarket subreddit. It provides a search engine and dynamic frontend that allows to quickly and efficiently search for adverts posted on the subreddit.

Next features:

  • Allow users to define alerts.
  • Work on a finer grained parsing mechanism.

FAQ

  • Where does the name Fleebmarket comes from ?
    Fleebmarket is kind of a contraction between « flea-market » and « keeb ».
  • Does Fleebmarket track me ?
    As of now, there is no tracker of any kind on the website. I do not intend to include any in the future either.
  • Is Fleebmarket RGPD compliant ?
    In the event that a reddit post containing personal information would find it's way on fleebmarket, you can request for it's deletion. Otherwise, since no personal data is hosted on fleebmarket, it should be RGPD compliant.

Ackownledgements

Fleebmarket is built atop great open-source software:
  • Django provides the central infrastucture of the service (along with various Django apps).
  • FastAPIDjango REST Framework is used to implement the API
  • React is used for the dynamic parts of the interface.
  • The django db runs on a PostgreSQL instance.
  • The text search functionality is provided by a Meilisearch instance.
  • Bulma provides thee CSS framework, along with django-bulma for integration into Django templates.
  • Uvicorn and nginx provide the web server facilities.
  • Everything is hosted on a single Rock64 board running Manjaro
